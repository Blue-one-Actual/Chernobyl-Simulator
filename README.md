# Chernobyl-Simulator

Dies ist eine vereinfachte, pädagogische Web‑Simulation eines Kontrollraums. Sie soll Lern‑ und Visualisierungszwecken dienen und KEINE echten Steuerungsmaßnahmen ersetzen.

Starten:

- Öffne die Datei [index.html](index.html) in einem Browser (lokal öffnen reicht).
- Bediene den Regler `Regelstab-Einzug`, aktiviere/deakti viere die Kühlung und nutze `Start`/`Stop`.
- `EMERGENCY SCRAM` fährt die Regelstäbe vollständig ein und stoppt den Reaktor.

Dateien:

- [index.html](index.html)
- [styles.css](styles.css)
- [app.js](app.js)

Hinweis: Die Simulation ist stark vereinfacht und rein demonstrativ.
